# Treasure Hunt Game

A campus-based 2D treasure hunt adventure game developed using the RayMob framework built on top of Raylib.  
Players explore a large interactive campus map, solve clue-based puzzles, discover treasures, avoid penalties, and progress through multiple levels.

---

## Features

- Interactive campus exploration
- Clue-based treasure hunt gameplay
- Multiple levels with increasing difficulty
- Reward system with coins and diamonds
- Nitro speed zones
- Responsive mobile joystick controls
- Camera panning and pinch zoom
- Save/Load game progress
- Dynamic clue generation system
- Touch and desktop input support
- RayMob framework integration
- Built using Raylib graphics framework

---

## Tech Stack

- C++
- Raylib
- Raymath
- RayMob Framework

---

## Gameplay Overview

Players navigate through different campus locations such as:

- Library
- Sports Complex
- Laboratory
- Mess
- Temple
- AI & DS School
- Faculty Quarters
- Restaurant
- PHC
- Liberal Arts School

At each location, players analyze clues and determine the correct destination to unlock treasures.

Wrong guesses increase penalties, while correct answers reward treasures, coins, and diamonds.

---

## Controls

### Mobile
- Virtual Joystick → Movement
- Tap Buttons → Interactions
- Pinch Gesture → Zoom
- Drag → Camera Pan

### Desktop
- Mouse Input for UI interaction
